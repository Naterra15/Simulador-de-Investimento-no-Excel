# Simulador-de-Investimento-no-Excel
Este projeto tem como objetivo a criação de uma ferramenta em Excel para simular investimentos imobiliários, calcular rendimento mensal, estimar dividendos ao longo do tempo, sugerir um valor ideal de investimento com base no salário e classificar o perfil de investidor (conservador, moderado, agressivo).
Funcionalidades
A planilha permite:
Inserir salário mensal
Sugerir um valor ideal de investimento com base em percentual da renda
Classificar o perfil do investidor (Conservador, Moderado, Agressivo)
Projetar crescimento do capital ao longo do tempo
